# Images Folder

Folder ini digunakan untuk menyimpan gambar-gambar dokumentasi proyek LMS Platform.

## Cara Menambahkan Gambar dari Canva

1. Buka link Canva: https://www.canva.com/design/DAG4q7E_jHs/UO-HywbWzhXkhgHyAAlPqA/edit
2. Klik tombol **"Share"** atau **"Download"**
3. Pilih format **PNG** atau **JPG**
4. Simpan gambar dengan nama: `lms-platform-preview.png`
5. Letakkan file di folder `images/` ini
6. Gambar akan otomatis tampil di README.md

## File yang Diharapkan

- `lms-platform-preview.png` - Preview utama LMS Platform dari Canva
